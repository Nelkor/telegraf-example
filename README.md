# An example of Telegram bot

## Environment

Create a `.env`-file from the `.env.example` one

## Docker

`docker-compose up -d`

## Commands

|                      |                  |
|----------------------|:-----------------|
| Install dependencies | `npm i`          |
| Run tests            | `npm run test`   |
| Check types          | `npm run ts`     |
| Lint                 | `npm run lint`   |
| Format               | `npm run format` |
| Build                | `npm run build`  |
| Run application      | `npm start`      |
|                      |                  |
